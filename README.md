# Simulador de Decisiones Financieras

Web app simple de Cocoo para evaluar si una persona puede sostener una decisión financiera importante antes de tomarla.

## Alcance

- Sin login
- Sin base de datos
- Sin pagos
- Sin descargas
- Cálculos en frontend

## Desarrollo

```bash
npm install
npm run dev
```

## Validación

```bash
npm run build
```
